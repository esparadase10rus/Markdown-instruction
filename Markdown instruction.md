## Списки

Что бы добавить не нумерованные списки,
необоходимо пункты выделить (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить не нумерованные списки,
необоходимо пункты выделить (-).
Например, вот так:
- Элемент 1
- Элемент 2
- Элемент 3

Что бы добавить не нумерованные списки,
необоходимо пункты выделить (+).
Например, вот так:
+ Элемент 1
+ Элемент 2
+ Элемент 3


Что бы добавить нумерованные списки,
необходимо пуктны просто пронумеровать.
Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
